## 1. Описание задачи

### Цель работы
Разработать программу для автоматизации расчёта расхода ткани и стоимости пошива одежды с использованием объектно-ориентированного программирования, модулей и пакетов в Python.

### Вариант задания
**Вариант 10 - Одежда**

| № | Вид одежды | Особенности |
|---|------------|-------------|
| 1 | **Пиджак** | Расход ткани: 2.2-2.7 м в зависимости от размера |
| 2 | **Брюки** | Расход ткани: 1.2-1.45 м в зависимости от размера |
| 3 | **Костюм-тройка** | Пиджак + Брюки + Жилет (дополнительно 0.8 м ткани) |

### Требования к программе

1. Создать **пакет**, содержащий **3 модуля** (по одному на каждый вид одежды)
2. Реализовать **графический пользовательский интерфейс** с возможностью:
   - Выбора типа одежды
   - Выбора размера (от 44 до 54)
   - Отображения результатов расчёта
3. Обеспечить **сохранение отчёта** в форматах:
   - `.docx` (Microsoft Word)
   - `.xlsx` (Microsoft Excel)
4. Оформить отчёт в файле `README.md`

### Исходные данные для расчётов

**Таблица расхода ткани по размерам:**

| Размер | Пиджак (м) | Брюки (м) | Жилет (м) |
|--------|------------|-----------|-----------|
| 44 | 2.2 | 1.2 | 0.8 |
| 46 | 2.3 | 1.25 | 0.8 |
| 48 | 2.4 | 1.3 | 0.8 |
| 50 | 2.5 | 1.35 | 0.8 |
| 52 | 2.6 | 1.4 | 0.8 |
| 54 | 2.7 | 1.45 | 0.8 |

**Финансовые параметры:**

| Параметр | Пиджак | Брюки | Жилет |
|----------|--------|-------|-------|
| Цена ткани за 1 метр | 800 руб | 800 руб | 800 руб |
| Стоимость работы за 1 метр | 1200 руб | 1000 руб | 1000 руб |
| Стоимость фурнитуры | 500 руб | 300 руб | 200 руб |

### Формулы расчёта
Расход ткани = FABRIC_PER_SIZE[размер] (или ближайший существующий размер)

Стоимость ткани = Расход ткани × Цена за метр (800 руб)

Стоимость работы = Расход ткани × Стоимость работы за метр

Фурнитура = фиксированная стоимость

ИТОГО = Стоимость ткани + Стоимость работы + Фурнитура
